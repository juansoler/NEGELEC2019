# Desarrollo de algoritmo “apriori” para encontrar de forma eficiente "conjuntos de ítems frecuentes“ en una cesta de la compra.
### Universidad de Almería. Grado en Ingeniería Informática.

Asignatura: Negocio Electrónico.

Profesor: José Antonio Torres Arriaza.

Alumno: Juan Soler Márquez.



## Introducción
El objetivo de la práctica es realizar una funcionalidad de una aplicación de negocio electrónico. 

Para ello hemos implementado el algorimo apriori para que realice tareas de asociación de un conjunto de datos. 

Además se ha desarrollado un generador de cestas de compra aleatorias con una serie de productos inventados.
